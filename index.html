科科紫微任務包產生器 v0.1

把文墨天機文字包整理成可交給小科的 Markdown / TXT。只整理，不排盤，不解盤。

純前端 本機處理 不保存 不接 API

1. 貼上資料SOURCE

0 字 建議上限 50,000 字

2. 選擇本次任務TASK

資料檢查CHECK

檢查只確認文字包是否「看起來有包含」必要段落，不判斷命盤正確性，也不補資料。

所有資料只在目前瀏覽器分頁處理。這個工具不使用外部字型、CDN、API、localStorage、Cookie 或資料庫。

3. 輸出任務包OUTPUT

尚未產生任務包。
