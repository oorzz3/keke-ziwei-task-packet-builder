# 科科命理任務包產生器 v0.3-alpha

這是一個獨立靜態工具，用來把科科手上的命理資料整理成可以交給小科解讀的 Markdown / TXT 任務包。

本工具是任務包產生器，不是命理計算器。

## 三種模式

### 紫微任務包

- 整理文墨天機紫微文字包
- 保留 v0.1.1 的欄位偵測、建議附圖、Markdown / TXT 輸出
- 適合今日、流日、流時、工作、財務、感情、健康、人際等任務

### 八字任務包

- 整理固定八字 TXT、分析日期與科科本次問題
- 可貼上文字，也可匯入本機 TXT
- 網站不推算今日干支、不判斷十神、不做八字排盤
- 今日干支、流日十神與能量解讀交給小科

### 合併對照包

- 整理八字候選日 / 八字摘要與紫微流日資料
- 產生八字 × 紫微合併對照任務包
- 交給小科比對一致點、反證與違和點
- 網站不判斷吉凶

## 共用功能

1. 貼上資料
2. 匯入本機 TXT（僅八字模式）
3. 選擇任務 / 目標
4. 填入科科本次問題
5. 欄位偵測
6. 建議附圖 / 補充資料
7. 選擇小科老師模式
8. 產生 Markdown / TXT 任務包
9. 一鍵複製
10. 下載 `.md` 或 `.txt`
11. 清除畫面資料

## 禁止事項

- 不做紫微排盤
- 不做八字排盤
- 不做八字日柱演算法
- 不做八字月度雷達自動計算
- 不做解盤
- 不做 OCR
- 不接 API
- 不保存資料
- 不寫死科科生日或八字資料
- 不使用外部資源
- 不使用瀏覽器儲存或 cookie
- 不把欄位偵測包裝成資料成立判斷
- 不輸出保證吉凶
- 不輸出投資買賣指令、醫療診斷或法律建議

## 百分比文案

若任務包提到百分比，意思是「注意力 / 作息配置比重」，不是事件發生機率。

## 資料安全

這個工具是純前端靜態網站。所有資料只在目前瀏覽器分頁處理，不上傳、不保存、不連接外部服務。
