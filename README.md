# 科科紫微任務包產生器 v0.1

這是一個獨立靜態工具，用來把科科從文墨天機取得的文字命盤包，整理成一份可以交給小科解盤的 Markdown / TXT 任務包。

## 定位

- 文墨天機：負責排盤
- 科科：貼上文字包、選擇問題、附上截圖
- 小丸工具：整理資料、檢查欄位、產生任務包
- 小科：依任務包與截圖解盤

## v0.1 功能

1. 貼上文墨天機文字包
2. 選擇分析主題
3. 檢查資料是否包含 App版本、安星碼、基本信息、命盤十二宮、流年、流月、流日
4. 若選流時，檢查流時 / 時曜
5. 顯示建議附上的截圖清單
6. 產生 Markdown / TXT 任務包
7. 一鍵複製全文
8. 下載 `.md` 或 `.txt`
9. 清除畫面資料

## 禁止事項

- 不做紫微排盤
- 不做解盤
- 不做 OCR
- 不接 API
- 不保存資料
- 不寫死科科生日資料
- 不接命盤宇宙站
- 不宣稱 calculated

## 資料安全

這個工具是純前端單頁 HTML。所有資料只在目前瀏覽器分頁處理，不使用外部 CDN、外部字型、API、localStorage、Cookie 或資料庫。
